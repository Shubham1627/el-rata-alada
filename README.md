# 🦇 EL RATA ALADA

Step into the Batcave of containers! EL RATA ALADA is a **Dockerized one-time chat app** built with:

- **API:** Node.js backend managing sessions & messages  
- **Frontend:** Nginx-powered web UI  
- **Redis:** Ephemeral session storage  

Spin up the app with **Docker Compose**, create one-time chat sessions, send messages, fetch history, and destroy sessions instantly.  
	$ docker compose up --build

Pre-requisite: 
	- Docker
